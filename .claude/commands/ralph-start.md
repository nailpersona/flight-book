# Ralph Start - Автоматичне виконання всіх stories

Виконуй stories з PRD.md в циклі, поки всі не будуть готові.

## Головний цикл

```
LOOP:
  1. Перевір .ralph-stop файл → якщо є, STOP
  2. Прочитай PRD.md
  3. Знайди першу невиконану story [ ]
  4. Якщо немає → ВСІ ГОТОВІ, STOP
  5. Виконай цю story
  6. Познач [x] в PRD.md
  7. Запиши в progress.txt
  8. Повтори LOOP
```

## Виконання кожної story

1. Прочитай потрібні файли
2. Виконай задачу (код, база даних, тести)
3. Перевір що працює
4. Познач `[x]` в PRD.md

## Зупинка

- Файл `.ralph-stop` існує → зупинись
- Всі stories `[x]` → повідом "✅ ВСІ STORIES ВИКОНАНО"
- Story блокується → повідом "❌ БЛОК: [причина]" і зупинись

## Звіт після кожної story

```
✅ Story N: [назва]
⏳ Залишилось: N stories
→ Продовжую...
```

## Важливо

- НЕ зупиняйся після однієї story
- Продовжуй цикл автоматично
- Стискай контекст між stories (залиш тільки: поточна story, ключові ID)
- Якщо контекст переповнений → короткий summary і продовжуй
